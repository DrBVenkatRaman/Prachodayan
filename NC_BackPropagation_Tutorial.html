Learning by BackPropagation
Tutorial

When the assigned weigths to the inputs do not produce satisfactory output, the weights need to be updated so that some correction is made for the improvement of the result. Backpropogation is the algorithm which shows the generalied idea of how to update the weights so that the neural network converges to produce satisfactory results. The algorithm is explained step by step below.

Backpropogation Algorithm:

Step 1: Set some initial weights to all the inputs of a neuron including bias.

Step 2: Perform summation and activation operations of the neuron.

Step 3: Verify the output:
    For supervised learning: check if whether it is equal to desired output or not.
    For unsupervised learning: ensure that it qualifies the performance metric or not.

Step 4: If the output is found not satisfactory, then update the weights according to any selected learning rule.
    w(t+1) = w(t) + δw
    'δw' is determined by the learning rule

    [In this experiment,
    threshold activation function with '0' as threshold value is used and
    the learning rule used is Perceptron Learning rule (Refer 'learning rules' topic for more info)]

Step 5: Repeat step 2 to step 4 with updated weights on entire training data till satisfactory performance is obtained.
